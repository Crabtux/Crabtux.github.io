3-1

下载下来的文件无后缀名，用 010editor 查看，发现是 rar 压缩包。
修改后缀名之后打开，解压其中的文件。

解压出来的文件仍然没有后缀名，用 010editor 查看，发现是 wireshark 流量包，打开即可。

追踪TCP流，在流5、6中发现了 flag.rar、1、2、3 四个文件(没错，后三个数字都是文件名，并且无后缀名)。
flag.rar 是个加密压缩文件，先放一边晾着。
1 是坏掉的 flag.rar。吃灰去吧你！
2 是个 base64 字符串，先放一边晾着。
3 是个 python 程序，使用的是AES-256加密，密文都写在程序里了。

猜想 3 解密出来的玩意就是 flag.rar 的密码，所以先研究它。
猜想 2 的 base64 解密成果就是 3 所需要的密文。
两个猜想都是正确的，压缩包解压后即可得到flag。

3-2

下载下来的 gif 文件有 4 帧，拼起来变成一个二维码。
扫描得到一个 16 进制串，扔到 010editor 里发现是 pyc 文件。
反编译获得 python 文件，运行 func2(strr,key) 即可得到 flag。
有一点值得注意：
运行 func2 的时候，运行环境不同可能会导致运行结果不同。
比如本文件在 Windows（Python3）下会输出乱码，而 Linux 环境（Python2）运行正常。
