我好累 vim好难

前言

这篇文章本质上是写给我自己看的，而我这个人又极其懒惰。
因此我自认为不需要刻意记忆的东西就不会写在里面。（比如说模式之间的切换等等）
一些我容易遗忘的东西我就会记在这里。
随着学习的深入 大概 会逐渐更新。
话说你要真想学vim，你也不该来看这篇文章啊

主要命令

移动

  • 基本移动：hjkl
  • 词：w、b、e
  • 行内：0、$
  • 屏幕： H、M、L
  • 整文件：G、g
  • 翻页：^F（下翻一页）、^B（上翻一页）
  • 滚一行：^E（下滚一行）、^Y（上滚一行）
  • 括号配对：%
  • 查找：/、？

编辑

  • O/o：在之上/之下插入行
  • x：删除字符
  • r：替换字符
  • s：替换字符并进入insert mode
  • d：删除（高级一点）
  • c：改变（高级一点）
  • y：复制
  • d：剪切
  • p：粘贴

命令行模式

  • :s/old/new：替换文件内容
  • :>：在行首添加制表符

多窗口模式

  • :new name 新建一个名为name的文件
  • :split(sp) name 横向分割
  • :vsplit(vs) name 纵向分割
  • :Ctrl+w+hjkl 多窗口切换
  • :(vertical) resize(res) 重置窗口大小
  • :q 退出当前窗口
  • :qa 退出所有窗口

一些奇怪的操作

  • 十六进制显示文件内容
    • bash中输入vim -b filename
    • vim中输入:%!xxd -g 1
    • 长得跟普通的十六进制编辑器差不多吧
